🎯 NEW: ZDiTM API PROXY INTEGRATION
───────────────────────────────────────────────────────────────────────────

Problem Solved:
  ❌ ZDiTM API blocks direct browser requests (CORS restrictions)
  ✅ Solution: Vercel serverless proxy function

How It Works:
  1. Browser requests: /api/zditm-departures?stops=Łucznicza,Tarczowa
  2. Vercel function receives request
  3. Function fetches from ZDiTM API (server-to-server, no CORS)
  4. Returns real-time departure data
  5. If ZDiTM is down, returns realistic simulated data

Files Changed:
  • /api/zditm-departures.js (100 lines) — Proxy function
  • Updated live.js — Fetch from proxy endpoint
  • Updated vercel.json — Configure serverless function

Data Source Indicator:
  • 🟢 ZDiTM (live) — Real-time data from API
  • 🟡 Symulowane — Fallback simulated data
